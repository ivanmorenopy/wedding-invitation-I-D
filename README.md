## Cómo publicarla en GitHub Pages

1. Sube los cambios a la rama `main`.
2. Ve a **Settings** del repositorio en GitHub.
3. En **Pages**:
	- **Source**: `Deploy from a branch`
	- **Branch**: `main` y carpeta `/ (root)`
4. Guarda y espera 1-2 minutos.

La URL quedará así:

`https://ivanmorenopy.github.io/wedding-invitation-I-D/`

## Editar y volver a publicar

Cambia `index.html`, haz commit y push a `main`. GitHub Pages redeploya automáticamente.
